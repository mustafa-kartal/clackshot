# Changelog

All notable changes to ClackShot will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-05-01

### Added
- Full screen, area selection, and window capture
- Screen recording via MediaRecorder with MP4 output
- Konva-based annotation editor (text, shapes, arrows)
- Face cam overlay during recording
- Automatic clipboard copy after capture
- PNG export to custom location
- System tray with background operation
- Global keyboard shortcuts (Cmd/Ctrl+Shift+3/4/5)
- macOS Screen Recording permission flow (TCC)
- Auto-update via GitHub Releases (electron-updater)
- Single instance lock and security guards
- Cross-platform builds: macOS (dmg, zip), Windows (nsis), Linux (AppImage, deb)
